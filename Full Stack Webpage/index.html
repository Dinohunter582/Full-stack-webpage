Why are text editors so great

Text editors are very good for programming and are not like other programming applications they have many things that set them apart from each other as well. We will go over some of the more well-known text editors.

what is a text editor

You may be asking yourself what exactly is a text editor. Well, a text editor is a program that edits plain text they have there own operating systems and software dev packages and can be used to change configuration files, document code, and source code. Keep in mind that plain text is very different from rich text, plain text is text made in a text editor and rich text is made in a word processor such as Microsoft Word.

The history of a text editor

Text editors were very simple they use to use punch cards you would put the punch card into a card reader they had no line breaks and could only fit 80 characters on one sheet. But once screens such as monitors became available the text editors made their way to screens and were known as screen editors but they were still very limited in what they could do on until they started experimenting with them more to make them what they are today.

Different types of text editors

Some text editors don't do all that much while others can take down pretty much anything you can throw at it. For example, mac os x has text edit which is a hybrid version of a text editor as it has features of a text editor and it has the features of a word processor as well but they must be turned on separately as it can not do both at the same time while it might not be the best it can still do a lot. Some text editors can edit massive files that simple text editors would struggle with these types of data sets such as log files or an entire database and can do it all in one file. While simple text editors cant edits massive files it can still read them in the computer's memory. some editors you can program even more than they already are to serve a specific purpose.

The features a text editor normally has

  • Find and Replace: text editors provide the ability to for text through a a group of files or just single ones, more advanced editors can use regular expressions to search and edit text or code.
  • Cut,Copy, and Paste: most text editors have a way to do this allowing you to place the same snippet of code in multiple places on the file or multiple files.
  • The abilit to hadle UTF-8 encoding.
  • Text formatting: Most text editors have the ability to perform basic visual formatting features such as line wrap, auto indentation, bullet list and many other things.
  • Undo and Redo: just like word processors, text editors can also do this how many edits a text editor will remember depends on which text editor you are using

Text editors have many different features like macro or procedure definition, multi view, multi file editing, etc. depending on which one you use they are all very different from each other. There are also specialised editors for example source code editors are text editors that normally have user programmable syntax highlighting and code navigation functions and more coding tools and keyboard macros that work just like they do in html editors.
